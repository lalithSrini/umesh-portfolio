ℹ️ How It Works

1. Edit any section in the sidebar.

2. Click 💾 Save Changes (or press Ctrl+S).

3. 🔴 Live Update: If your portfolio is open in another tab, it automatically re-renders — no refresh needed!

4. Click 🔍 Preview Portfolio in the sidebar to open and auto-refresh the portfolio tab.

5. Click ⬇ Export JSON → replace portfolio-data.json in your project folder to make changes permanent for all visitors.
